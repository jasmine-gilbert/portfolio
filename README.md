# Portfolio BTS SIO (SISR)

Petite structure de départ pour un portfolio BTS SIO (option SISR) avec thème bleu-gris clair.

**Fonctionnalités ajoutées**
- Menu complet avec dropdowns (Projets, Veilles, À propos)
- Page dédiée "Mon entreprise" avec informations et missions animées
- Fond d'écran animé à effet type informatique qui suit la souris sur la page d'accueil
- Sections animées (info-box) et design responsive

Fichiers présents :
- index.html (page d'accueil vide avec fond animé)
- mon-entreprise.html (page de l'entreprise Sauvegarde 13)
- css/style.css (styles + animations + canvas)
- js/main.js (navigation mobile + animation canvas)

Pour voir le site, ouvrez `index.html` dans votre navigateur (double-clic ou via un serveur local).
